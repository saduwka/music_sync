# iPod Music Sync

Скрипт для автоматической синхронизации плейлиста YouTube Music с вашим iPod (или другим устройством на iOS с Jailbreak) через SSH.

## Возможности

- 📥 **Автоматическое скачивание:** Загружает новые треки из указанного плейлиста YouTube Music.
- 🎵 **Конвертация:** Автоматически конвертирует аудио в формат M4A (256kbps) с метаданными и обложками.
- 🚀 **Беспроводная синхронизация:** Переносит файлы на iPod через SCP/SSH.
- 📱 **Библиотека iOS:** Использует `gimport` для автоматического добавления треков в медиатеку Music.app без iTunes.
- 🔔 **Уведомления:** Отправляет отчеты о работе в Telegram.
- 💾 **Отслеживание состояния:** Запоминает уже скачанные треки, чтобы не загружать их повторно.

## Требования

- **Python 3.7+**
- **yt-dlp:** Для скачивания музыки.
- **ffmpeg:** Для обработки аудио.
- **iPod/iPhone с Jailbreak:** Должны быть установлены `OpenSSH` и `Gremlin` (или `gimport`).
- **Библиотеки Python:** `requests`, `python-dotenv`.

## Установка

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/youruser/music_sync.git
   cd music_sync
   ```

2. Установите зависимости:
   ```bash
   pip install requests python-dotenv
   ```

3. Настройте окружение:
   Создайте файл `.env` (или скопируйте из `.env.example`) и заполните настройки:
   ```bash
   cp .env.example .env
   nano .env
   ```

4. Убедитесь, что у вас настроен SSH-доступ к iPod по ключам (чтобы не вводить пароль каждый раз).

## Использование

Просто запустите скрипт:
```bash
python sync.py
```

Для автоматизации можно добавить скрипт в `crontab`.

## Настройка .env

- `PLAYLIST_URL`: Ссылка на ваш плейлист YouTube Music.
- `IPOD_IP`: IP-адрес вашего iPod в локальной сети.
- `IPOD_USER`: Пользователь SSH (обычно `root`).
- `IPOD_MUSIC_PATH`: Временная папка на iPod для загрузки.
- `TELEGRAM_BOT_TOKEN`: Токен вашего бота (через @BotFather).
- `TELEGRAM_ADMIN_ID`: Ваш ID в Telegram.

## Структура проекта

- `sync.py`: Основной скрипт.
- `state.json`: База данных скачанных треков.
- `downloads/`: Временная папка для загрузки.
- `ipod_ready/`: Папка с готовыми к отправке файлами.
- `logs/`: Логи работы.
